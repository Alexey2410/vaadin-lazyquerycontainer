Lazy Query Container is a Vaadin framework addon which provides lazy loading data container implementation for Vaadin tables. Lazy Query Container is read only container and data modifications are designed to be done through a business delegate. The QueryFactory and Query interfaces are designed to be application specific and should apply the business delegate for querying data from business services.

Lazy Query Container supports the following features

  • Lazy Loading of Items
  • Loading Items in Batches
  • Item Caching with Limited Cache Size
  • Sorting
  • Changing Item Set
  • Changing Property Set
